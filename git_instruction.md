# Подсказка по Git

Создание репозитория
```sh
git init
```

Команда 
```sh
git add
```
позволяет **добавить файл в репозиторий**.


Делаем коммит:
```sh
git commit -m "Message"
```
>Вместо *Message* нужно подставить текст, который опишет основные изменения в этой версии файла.

git log
git log --oneline
git checkout
